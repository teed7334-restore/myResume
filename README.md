# myResume
我的履歷生成工具

## 介紹
可以依你輸入的內容，透過OpenAI生成對應的履歷簡歷，好比

中文履歷

```
尊敬的家裏蹲股份有限公司招聘負責人：

我是Peter Cheng，我是一位有15年軟體工程師經驗的男性應徵者，我有豐富的專業技能，包括PHP，Python，Node，Golang，Solidity，Java等，並且我不排斥擔任管理職。

我曾經參與過跨國電子商城開發、多店家分潤商城開發、物流系統開發、O2O商城開發、礦場管理系統開發、算力合約購買網站開發、雲簽証系統開發、電子發票系統開發和自動下單投資系統開發等項目。

我對於家裏蹲股份有限公司的職位表示非常感興趣，我會努力把握機會，發揮我的專業技能，為公司創造更大的價值。此外，我也希望在家工作，減少出行時間，為家庭提供更多的時間。

如果您對我的簡歷感興趣，請隨時與我聯繫，我的電子郵件地址是teed7334@gmail.com，手機號碼是0932179286。

我期待著有機會與您談話，以便更進一步地了解我的技能和能力。

謝謝您的考慮！

有敬意， Peter Cheng
```
